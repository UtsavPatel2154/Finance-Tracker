# 💰 Finance Tracker System

## 📌 Project Description

The Finance Tracker System is a web application developed using PHP and MySQL.
It helps users manage their personal finances by tracking income, expenses, budget, and savings.
The system stores financial data in a database and shows reports to help users understand their financial activity.

## 🚀 Features

* Add and manage income
* Add and track expenses
* Manage monthly budget
* Set savings goals
* View financial reports
* Clear stored financial data

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL

## 📂 Project Structure

FinanceTracker/
│
├── index.php        - Home page
├── income.php       - Income tracking page
├── expense.php      - Expense tracking page
├── budget.php       - Budget management page
├── saving.php       - Savings goals page
├── report.php       - Financial report page
└── database.sql     - Database structure

## ⚙️ How to Run the Project

1. Install XAMPP or any local server.
2. Copy the project folder into the **htdocs** folder.
3. Start **Apache** and **MySQL** from XAMPP.
4. Open **phpMyAdmin**.
5. Create a database named **financetracker**.
6. Import the **database.sql** file into the database.
7. Open your browser and go to:
   http://localhost/FinanceTracker
